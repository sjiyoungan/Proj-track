Usage Examples
Current Usage: Your tracker dropdown uses ChevronDown which is perfect for dropdown menus.

Alternative Options for Dropdowns:
• ChevronDown - Current choice (perfect for dropdowns)
• ChevronsDown - Double chevron for "show more" or "expand all"
• ArrowDown - More pointed arrow for navigation
• CaretDown - Smaller, more subtle indicator
• Triangle - Simple triangle shape

Other Arrow Uses:
• ArrowUpDown - For sortable columns
• ArrowsUpDown - For drag-and-drop indicators
• RotateCw - For refresh/reload buttons
• SkipForward - For pagination or navigation
